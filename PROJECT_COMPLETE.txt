🔧 BUGS CRITIQUES CORRIGÉS

[✅] BUG 1:  MetaMask Transaction Encoding
     Problem: Encoding hex invalide, pas de fonction selector
     Fix:     Utilisé viem.encodeFunctionData() avec ABI
     Impact:  Transactions MetaMask maintenant valides

[✅] BUG 2:  Backend Fixed Account Signing
     Problem: Backend signait avec hardhat deployer account
     Fix:     Supprimé backend signing, frontend signe via MetaMask
     Impact:  Vraies adresses étudiants enregistrées blockchain

[✅] BUG 3:  No STUDENT_ROLE Assignment
     Problem: Pas de mécanisme pour assigner rôle aux étudiants
     Fix:     Ajouté requestStudentRole() auto-service dans RoleManager
     Impact:  Étudiants peuvent demander eux-mêmes le rôle

[✅] BUG 4:  Document Import Error
     Problem: documentRegistryContract non exporté de client.cjs
     Fix:     Utilisé addresses.documentRegistry + documentRegistryAbi
     Impact:  Route upload document maintenant fonctionnelle

[✅] BUG 5:  Announcement Filter Logic
     Problem: Filtre groupe "ALL" montrait UNIQUEMENT "ALL" annonces
     Fix:     Supprimé else if logiquement cassé
     Impact:  Étudiants groupe "ALL" voient toutes les annonces

[✅] BUG 6:  Missing getAcknowledgments()
     Problem: Pas de fonction contrat pour récupérer signatures
     Fix:     Ajouté getAcknowledgments(announcementId) view
     Impact:  Professeur peut voir qui a signé

[✅] BUG 7:  No Signature Retrieval API
     Problem: Pas d'endpoint pour professeur voir signatures
     Fix:     Créé GET /api/acknowledge/announcement/:id
     Impact:  Professor UI affiche vraies données

[✅] BUG 8:  Incomplete Contract ABIs
     Problem: ABIs manquaient getAcknowledgments, getCount, isStudent, etc
     Fix:     Complété toutes les fonctions view manquantes
     Impact:  Backend peut lire toutes les données contrats

[✅] BUG 9:  Professor Signature Tracking UI
     Problem: Affichait placeholder au lieu vraies données
     Fix:     Intégré API call pour afficher signatures réelles
     Impact:  Professeur voit liste complète avec timestamps
